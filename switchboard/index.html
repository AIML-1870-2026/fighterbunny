ℹ️ Why can't I fetch models automatically sometimes?

Browsers enforce a security rule called CORS (Cross-Origin Resource Sharing) that controls which websites can talk to which APIs. Anthropic's API is designed to be called from servers, not directly from browser tabs — so depending on your setup, the model discovery call may be blocked.

This is normal and expected. If discovery fails, the app quietly falls back to a built-in model list. Your prompts will still work fine — CORS only affects the model list fetch, not the completions endpoint.

If you're running this locally via a dev server or proxy, discovery will likely work. If you're opening the HTML file directly, it may not — and that's okay.
